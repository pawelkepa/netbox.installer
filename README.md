# netbox.installer
Auto install netbox in ubuntu server 18.04 and 20.04

## How to install ?
```
# su - root
# git clone https://github.com/iman-homayouni/netbox.installer
# cd netbox.installer/
# bash netbox.installer.sh -h
```
